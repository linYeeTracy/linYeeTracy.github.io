# 风格

不局限于具体风格

# 色调

上次提到的灰色色调或暖色（米色）色调皆可(乳胶漆)，另外，考虑墙面是否加入部分砖墙墙纸

# 布局

  1. 本布局按上次所提不变，原则上体现开放式，大空间感，通透感，个人还是希望加入入户走廊错层分割功能区域，后期可改成客房
  2. 加入阳台错层结构，阳台可以用作看书聊天工作区域

# 光源

1.暂定不要主灯，光源分散，以射灯，筒灯，隐藏式灯带实现。

# 期望

  1. 突出整屋的空间感，但不要空旷
  2. 加入3D延伸画/镜面元素让整屋视觉效果更大
  3. 中西分厨，希望开放式+吧台，岛台与餐桌结合构成整个用餐区域
  4. 入户走廊到阳台那块L型区域希望可以加入错层（当然，如果错层不好看且不合理完全可以不要，视具体情况而定）
  5. 区域划分可通过推拉门or窗帘
  6. 梳妆台最好可以在浴室，也可以在主卧
  7. 书房里要有收藏架的位置

# 生活习惯

  • 互联网行业从业者
  • 喜欢晒太阳，喜欢有空间感，可能会摆很多装饰物（大件小件都有）在室内
  • 电子设备较多
  • 在吃饭的时候喜欢看视频看电视
  • 喜欢吧台、拱门风格
  • 书房书桌要比较大，可能一个桌子上摆3个显示器，同时两个人一起工作
  • 5年内2-4人居住，10年内2-5人居住
  • 浴缸是硬性需求
  • 鞋子储物需求大

# 餐厨

  1. 不分中西厨了，中西厨都在中厨的那个房间，不用隔一堵墙区分中西厨，厨房仍然以开放式为主。 另外由于厨房那块有个承重墙打不掉，但是我想让餐厨空间能够合为一体，有个更开放的想法--将除了承重墙外其他墙体都打掉，将台面延伸出去作为吧台，但是这样可能会丧失一部分储物空间，但这部分储物空间我觉得不是必须的。具体想听下设计师的意见。

  1. 之前的西厨区域结合岛台，作为备餐区、柜区、洗衣区的结合。

  2. 橱柜结构 不过这块好坏与否需要设计师的建议。

# 书房（入户走廊）

  1. 书房由折叠房移至入户走廊，本身没有太多的藏书需求，可以将部分储物功能移入书房：

  1. 能不能留个从客厅直接进入书房的通道，而不用从阳台那侧绕过去，客厅部分已说明

  2. 照明用射灯或一排筒灯

# 客房兼娱乐室

这个房间还是以开放通透为主，这次修改中修改为榻榻米+升降桌结构，榻榻米结合储物功能，父母的意思是门口旁边的展示柜改成储物柜，屋内也要有一定的储物能力。

  1. 推拉门想要这种黑网格框风格的推拉门：

  1. 这个房间是有个飘窗的，是否可以降低飘窗的高度或者抬升榻榻米高度使其一致? 这样榻榻米面积会大点。

# 主卧

  1. 主卧的卫生间，最好将浴缸放在飘窗旁边以便利用飘窗空间。

  2. 保证浴室空间的情况下，衣帽间可能会略显小，衣帽间如果往阳台方面延伸可不可行（压缩主卧空间提升衣帽空间）。

  3. 妆台最好可以在主卫，主卫放不下放卧室也行。

  4. 想把主卧次卧两个卧室的门改成拱门式设计，是否可行？

# 次卧

暂无意见。

# 浴室

  1. 不用嵌入式浴缸是否可行？感觉嵌入式浴缸略占面积(缺点是容易藏污纳垢)。
  2. 最好将浴缸放在飘窗旁边以便利用飘窗空间。
